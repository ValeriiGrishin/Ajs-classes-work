Описание проекта:

Реализация иерархии классов игровых персонажей на JavaScript. Базовый класс Character наследуется шестью дочерними классами: Bowman, Swordsman, Magician, Daemon, Undead, Zombie.

Установка зависимостей:

npm install

Запуск тестов:

npm test

Запуск тестов с отчетом о покрытии:

npm run coverage:

Проверка стиля кода:

npm run lint

Сборка проекта:

npm run build
